# TicTacToe

### It uses Python 3 and an optimisation algorithm to play against you

### How to use it?

 - Make sure you have Python 3 and pygame installed 
   (https://www.pygame.org/wiki/GettingStarted)

 - Run `python3 runner.py`

 - Enjoy!
